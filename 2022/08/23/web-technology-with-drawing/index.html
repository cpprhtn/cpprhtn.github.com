그림으로 배는 웹 구조

영진닷컴에서 새로 출간한 그림으로 배우는 웹 구조 입니다.

저는 이전에 그림으로 배우는 C++ 시리즈를 읽어보았고, 좋은기억으로 남아있었기에 기대를 가지고 읽게되었습니다.

우선 이 책을 읽어보면 매우 기본적인 용어들부터 어렵지만 중요한 핵심용어까지 웹의 전반적인 개념을 다루고 있었습니다.

단순히 책을 읽기만 하는게 아니라 따라 해 볼 수 있는 페이지도 제공되어 있었습니다.

고전적인 웹의 구조 뿐만 아니라, 반응형 웹, SNS, 나아가 클라우드 서비스와 docker, k8s(kubernetes) 등의 현 시점에서의 웹 구조까지 다양하게 설명해주고 있었습니다.

또한 웹 서비스를 구축한다면 신경써야할 부분들을 한 Chapter에서 다양하게 설명해주고 있으며, 따로 공부하지 않았으면 몰랐을 보안적인 부분까지 다루는 책이였습니다.

이 책을 읽고나니 그동안 만들어본 웹에 대해 고민해보게 되었습니다.

저는 AI 개발자이며, 보통은 웹을 다룰일이 잘 없지만 해커톤이나 Toy project 등을 준비하면 본인이 만든 결과물을 웹이나 앱에 띄워서 효과적으로 보여줘야 할 일이 종종 생겼습니다.

지금까지는 웹을 결과를 띄우기 위한 수단으로만 생각했었고 따로 공부할 생각조차 해보지 않았었는데, 이번 기회를 통하여 웹의 전반적인 틀을 알게되어 너무 좋은 기회가 되었습니다.

요즘 도커 위주로 작업을 하고있는데 웹도 도커위에 올려서 깔끔하게 배포 하는 형태로 응용하여 사용해야겠습니다 :)

본 포스트는 영진닷컴 신간 서평 이벤트 차원에서 작성된 글입니다.
